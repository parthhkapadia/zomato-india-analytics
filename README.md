# zomato-india-analytics
An interactive data dashboard analyzing 200,000+ restaurant listings across India using the Zomato dataset. Built with vanilla HTML, CSS, and JavaScript — fully static and hosted on GitHub Pages.
Tech Stack

- HTML / CSS / JavaScript
- Chart.js — data visualizations
- PapaParse — CSV parsing
- GitHub Pages — deployment

---

## Features

- 5 interactive charts with real insights
- Filter by city and cuisine type
- Fully responsive dark UI
- Zero dependencies, no frameworks

---

## Dataset

Source: [Zomato India Dataset — Kaggle](kaggle.com/datasets/ngokulakannan/zomato-india-restaurants2-lakh-restaurants-data)  
Size: 200,000+ restaurant listings  
Fields used: City, Cuisine, Rating, Price Range, Online Delivery, Votes
